Bab 1: Dasar-Dasar JavaScript

Mengenal JavaScript, cara menjalankan, dan menampilkan output.

1.1 Pengenalan JavaScript

Bayangkan HTML sebagai kerangka rumah, CSS sebagai cat dan dekorasi, dan JavaScript sebagai penghuni yang membuat rumah menjadi hidup. JavaScript memungkinkan halaman web bereaksi terhadap interaksi pengguna.

Mengapa perlu JavaScript? Karena pengguna ingin pengalaman interaktif: tombol diklik, gambar bergeser, validasi form, dll. JavaScript berjalan di browser (client-side), jadi tidak perlu instalasi tambahan.

MARKICOB 1: Coba JavaScript di Console

Buka browser, tekan F12, pilih tab Console. Ketik kode berikut dan lihat hasilnya:

console.log("Halo, ini percobaan pertama!");
alert("Selamat datang di JavaScript!");
let nama = "Budi";
console.log("Nama saya " + nama);

Perhatikan: console.log menampilkan teks di console, alert memunculkan pop-up. Variabel nama menyimpan data dan bisa digunakan ulang.

Apa yang bisa dipelajari? Cara dasar menulis dan menjalankan JavaScript, serta penggunaan variabel sederhana.

1.2 Cara Menjalankan JavaScript

1.2.1 Melalui Console Browser

Tekan F12 → tab Console, lalu ketik kode JavaScript. Cocok untuk eksperimen cepat.

console.log("Halo, dunia!");
alert("Ini alert dari console");

1.2.2 Embedded / Internal JavaScript

Kode ditulis dalam tag <script> di file HTML.

<!DOCTYPE html>
<html>
<head><title>Embedded JS</title></head>
<body>
    <script>
        alert("Halo dari embedded JS!");
    </script>
</body>
</html>

1.2.3 Inline JavaScript

Kode ditulis langsung di atribut event HTML, misal onclick.

<button onclick="alert('Tombol diklik!')">Klik</button>
<p onmouseover="this.style.color='red'" onmouseout="this.style.color='black'">Arahkan mouse</p>

1.2.4 External JavaScript

Kode dipisah ke file .js, dipanggil dengan <script src="..."></script>.

<!-- index.html -->
<script src="script.js"></script>
// script.js
alert("File eksternal siap!");

MARKICOB 2: Praktik Embedded dan Inline

Coba buat file HTML dengan kode berikut. Simpan dan buka di browser. Amati apa yang terjadi ketika tombol diklik dan mouse diarahkan.

<!DOCTYPE html>
<html>
<head><title>Percobaan MARKICOB</title></head>
<body>
    <h1>Selamat Datang di Percobaan</h1>
    <button onclick="alert('Kamu mengklik tombol!')">Klik Aku</button>
    <p onmouseover="this.style.backgroundColor='yellow'" onmouseout="this.style.backgroundColor='white'">Arahkan mouse ke sini</p>
    <script>
        console.log("Halaman dimuat!");
    </script>
</body>
</html>

Latihan ini membantu memahami event onclick, onmouseover, dan penggunaan console.

1.3 Menampilkan Output di JavaScript

MetodeKegunaan
console.log()Menampilkan pesan di console (debugging)
alert()Kotak dialog pop-up
document.write()Menulis langsung ke dokumen (hati-hati)
innerHTMLMengubah konten elemen HTML

Contoh kode:

<div id="tempat"></div>
<script>
    console.log("Pesan di console");
    alert("Ini alert!");
    document.write("<p>Ditulis dengan document.write()</p>");
    document.getElementById("tempat").innerHTML = "<b>innerHTML</b>";
</script>

MARKICOB 3: Mengubah Konten dengan innerHTML

Buat file HTML berikut dan buka di browser. Klik tombol untuk melihat perubahan.

<!DOCTYPE html>
<html>
<head><title>Percobaan innerHTML</title></head>
<body>
    <h2 id="judul">Judul Awal</h2>
    <button onclick="ubahJudul()">Ubah Judul</button>
    <script>
        function ubahJudul() {
            document.getElementById("judul").innerHTML = "Judul telah diubah!";
        }
    </script>
</body>
</html>

Perhatikan bagaimana fungsi ubahJudul() menggunakan innerHTML untuk mengubah teks heading. Ini adalah dasar manipulasi DOM.

Latihan 1.2
  1. Buat file HTML dengan embedded JS yang menampilkan alert "Halo dari embedded".
  2. Buat tombol dengan inline JS yang mengubah teks sebuah paragraf menjadi "Teks berubah".
  3. Buat file HTML dan external JS yang menampilkan alert dan mengubah teks sebuah paragraf.
Latihan 1.3

Buat halaman HTML dengan sebuah paragraf kosong ber-id "demo". Gunakan innerHTML untuk mengisi paragraf tersebut dengan teks "Halo dari JavaScript". Tampilkan juga alert dan console.log dengan pesan bebas.
