Bab 6: Manipulasi DOM

Mengakses dan mengubah elemen HTML, menambah/menghapus elemen, dan menangani event.

MARKICOB – Coba Sendiri!

Buka console browser dan jalankan kode berikut pada halaman web apa pun. Lihat bagaimana JavaScript bisa mengubah halaman.

// Pilih elemen body
let body = document.body;
// Ubah warna latar
body.style.backgroundColor = "lightblue";
// Buat elemen baru
let p = document.createElement("p");
p.textContent = "Paragraf baru dibuat dari console!";
body.appendChild(p);

Ini adalah contoh manipulasi DOM secara langsung.

6.1 Memilih Elemen HTML

Sebelum memanipulasi, kita harus memilih elemen yang akan diubah. Berikut metode-metodenya:

MetodeDeskripsi
getElementById("id")Memilih satu elemen berdasarkan id.
getElementsByClassName("class")Memilih semua elemen dengan class tertentu (hasil seperti array).
getElementsByTagName("p")Memilih semua elemen dengan tag tertentu.
querySelector(".class")Memilih elemen pertama yang cocok dengan selektor CSS.
querySelectorAll("p")Memilih semua elemen yang cocok dengan selektor CSS.
 
let judul = document.getElementById("judul");
let paragraf = document.getElementsByClassName("teks");
let pertama = document.querySelector(".teks");
Latihan 6.1

Buat halaman dengan heading ber-id "header" dan tiga paragraf ber-class "isi". Gunakan berbagai metode seleksi dan tampilkan di console apa yang didapat.

<!DOCTYPE html>
<html>
<head><title>Latihan Seleksi</title></head>
<body>
    <h1 id="header">Heading</h1>
    <p class="isi">Paragraf 1</p>
    <p class="isi">Paragraf 2</p>
    <p class="isi">Paragraf 3</p>
    <script>
        console.log(document.getElementById("header"));
        console.log(document.getElementsByClassName("isi"));
        console.log(document.getElementsByTagName("p"));
        console.log(document.querySelector(".isi"));
        console.log(document.querySelectorAll(".isi"));
    </script>
</body>
</html>

6.2 Mengubah Konten dan Gaya

Setelah memiliki elemen, kita bisa ubah isi dan tampilannya.

Mengubah konten dengan innerHTML atau textContent

let judul = document.getElementById("judul");
judul.innerHTML = "Judul Baru"; // bisa menyertakan tag HTML
judul.textContent = "Judul Baru"; // hanya teks

Mengubah gaya dengan style

judul.style.color = "red";
judul.style.fontSize = "30px";
judul.style.backgroundColor = "yellow";
Latihan 6.2

Buat sebuah div dengan teks "Awal". Sediakan tombol yang mengubah warna latar belakang div menjadi biru dan teksnya menjadi "Berubah".

<div id="kotak" style="width:200px;height:100px;background:lightgray;">Awal</div>
<button onclick="ubah()">Ubah</button>
<script>
    function ubah() {
        let kotak = document.getElementById("kotak");
        kotak.style.backgroundColor = "blue";
        kotak.innerHTML = "Berubah";
    }
</script>

6.3 Menambah dan Menghapus Elemen

Menambah elemen baru

Langkah: createElement, isi konten, lalu appendChild.

let liBaru = document.createElement("li");
liBaru.textContent = "Item baru";
document.getElementById("list").appendChild(liBaru);

Menghapus elemen

Gunakan remove() atau removeChild().

let hapus = document.getElementById("item");
hapus.remove(); // langsung hapus

// Atau hapus dari induk
let induk = document.getElementById("list");
let item = document.getElementById("item");
induk.removeChild(item);
Latihan 6.3

Buat daftar tugas sederhana: ada input teks dan tombol "Tambah". Setiap tugas ditambahkan sebagai item list. Setiap item list juga punya tombol "Hapus" yang menghapus item tersebut.

<!DOCTYPE html>
<html>
<head><title>Todo List</title></head>
<body>
    <h2>Todo List</h2>
    <input type="text" id="tugas">
    <button onclick="tambah()">Tambah</button>
    <ul id="list"></ul>
    <script>
        function tambah() {
            let input = document.getElementById("tugas");
            let teks = input.value.trim();
            if (teks === "") return;
            let li = document.createElement("li");
            li.textContent = teks;
            let btnHapus = document.createElement("button");
            btnHapus.textContent = "Hapus";
            btnHapus.onclick = function() {
                this.parentElement.remove();
            };
            li.appendChild(btnHapus);
            document.getElementById("list").appendChild(li);
            input.value = "";
        }
    </script>
</body>
</html>

6.4 Event Handling dengan addEventListener

addEventListener adalah cara modern untuk menangani event. Bisa menambahkan banyak listener, mudah dihapus.

let tombol = document.getElementById("tombol");
tombol.addEventListener("click", function() {
    alert("Tombol diklik!");
});

// Fungsi terpisah
function tampilkan() {
    console.log("Event terjadi!");
}
tombol.addEventListener("mouseover", tampilkan);

Event yang umum: click, mouseover, mouseout, keydown, keyup, dll.

Latihan 6.4

Buat kotak berwarna (div). Saat mouse masuk ke kotak, warnanya berubah. Saat mouse keluar, kembali ke warna semula. Gunakan addEventListener.

<div id="box" style="width:100px;height:100px;background:blue;"></div>
<script>
    let box = document.getElementById("box");
    box.addEventListener("mouseover", function() {
        this.style.backgroundColor = "red";
    });
    box.addEventListener("mouseout", function() {
        this.style.backgroundColor = "blue";
    });
</script>

MARKICOB – Eksperimen DOM Interaktif

Coba buat halaman dengan form sederhana yang menampilkan teks yang diketik secara real-time.

<input type="text" id="input" placeholder="Ketik sesuatu...">
<p id="output"></p>
<script>
    let input = document.getElementById("input");
    let output = document.getElementById("output");
    input.addEventListener("keyup", function() {
        output.textContent = "Kamu mengetik: " + this.value;
    });
</script>

Modifikasi agar hurufnya otomatis menjadi kapital.
