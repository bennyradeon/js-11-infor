Bab 2: Variabel, Tipe Data, dan Operator

Menyimpan data dengan variabel, mengenal tipe data, dan menggunakan operator.

2.1 Variabel

Variabel adalah tempat menyimpan data di memori. Ibarat kotak berlabel yang bisa diisi dan diubah isinya. Mengapa perlu? Untuk menyimpan data seperti nama, umur, skor, dll.

Cara deklarasi:

  • let – untuk data yang bisa berubah. Contoh: let umur = 16;
  • const – untuk data tetap (konstan). Contoh: const PI = 3.14;
  • var – cara lama, sebaiknya dihindari.

Jika mencoba mengubah const, akan error.

let umur = 16;
umur = 17; // diperbolehkan
const PI = 3.14;
// PI = 3.14159; // error!

MARKICOB 1: Bermain dengan Variabel

Buat file HTML dan coba kode berikut. Lihat hasilnya di console dan di halaman.

<!DOCTYPE html>
<html>
<head><title>Percobaan Variabel</title></head>
<body>
    <div id="output"></div>
    <script>
        let nama = "Ani";
        let umur = 15;
        const kelas = "XI";
        let teks = "Nama: " + nama + "<br>Umur: " + umur + "<br>Kelas: " + kelas;
        document.getElementById("output").innerHTML = teks;
        console.log("Nama:", nama, "Umur:", umur);
    </script>
</body>
</html>

Perhatikan bagaimana kita menggunakan let dan const. Coba ubah nilai umur dan lihat perubahannya.

2.2 Tipe Data

Setiap data punya jenis. Kita perlu tahu jenisnya agar bisa mengolah dengan benar.

TipeContohPenjelasan
String"Halo", 'Belajar'Teks, diapit kutip.
Number10, 3.14Angka bulat atau desimal.
Booleantrue, falseNilai kebenaran.
NullnullSengaja dikosongkan.
Undefinedlet x;Belum diisi.

Cek tipe dengan typeof.

let nama = "Budi";
console.log(typeof nama); // "string"

MARKICOB 2: Mengecek Tipe Data

<script>
    let a = "Halo";
    let b = 123;
    let c = true;
    let d;
    let e = null;
    console.log(typeof a, typeof b, typeof c, typeof d, typeof e);
    // string number boolean undefined object (null adalah object karena bug lama)
</script>

Amati hasilnya di console. Perhatikan bahwa typeof null mengembalikan "object" (ini adalah keanehan JavaScript).

2.3 Operator

Aritmatika

OperatorNamaContoh
+Penjumlahan5+2=7
-Pengurangan5-2=3
*Perkalian5*2=10
/Pembagian10/2=5
%Modulus (sisa bagi)10%3=1
**Pangkat2**3=8

Perbandingan

OperatorArtiContoh
==Sama nilai5=="5" true
===Sama nilai dan tipe5==="5" false
!=Tidak sama nilai5!=3 true
!==Tidak sama nilai/tipe5!=="5" true
>Lebih besar8>5 true
<Lebih kecil2<1 false
>=Lebih besar/sama5>=5 true
<=Lebih kecil/sama4<=3 false

Logika

OperatorNamaContoh
&&ANDtrue && false = false
||ORtrue || false = true
!NOT!true = false

MARKICOB 3: Menggunakan Operator

<script>
    let a = 10, b = 3;
    console.log("Penjumlahan:", a + b);
    console.log("Sisa bagi:", a % b);
    console.log("Apakah a > b?", a > b);
    console.log("Apakah a === '10'?", a === '10');
    let umur = 17;
    let punyaKTP = true;
    console.log("Boleh bikin SIM?", umur >= 17 && punyaKTP);
</script>

Perhatikan perbedaan antara == dan ===. Operator logika sangat berguna untuk pengambilan keputusan.

Latihan 2.1

Buat program dengan let untuk menyimpan nama dan nilai matematika, dan const untuk menyimpan nama sekolah. Tampilkan semua data ke halaman menggunakan innerHTML.

Latihan 2.2
