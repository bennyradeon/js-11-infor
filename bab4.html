Bab 4: Array

Menyimpan banyak data dalam satu variabel, menambah, menghapus, dan mengolah array.

MARKICOB – Coba Sendiri!

Jalankan kode berikut di console browser. Amati bagaimana array bekerja.

// Membuat array dan mengakses elemen
let buah = ["Apel", "Mangga", "Jeruk"];
console.log(buah[0]); // Apel
console.log(buah[1]); // Mangga
console.log(buah.length); // 3

// Menambah elemen dengan push
buah.push("Pisang");
console.log(buah); // ["Apel", "Mangga", "Jeruk", "Pisang"]

// Menghapus elemen terakhir dengan pop
let terhapus = buah.pop();
console.log(terhapus); // Pisang
console.log(buah); // ["Apel", "Mangga", "Jeruk"]

Modifikasi kode: coba tambah di indeks tertentu, atau gunakan perulangan untuk menampilkan semua.

4.1 Apa itu Array?

Array adalah struktur data yang digunakan untuk menyimpan banyak nilai dalam satu variabel. Setiap nilai disebut elemen, dan diakses menggunakan indeks yang dimulai dari 0.

Mengapa perlu array? Misal kita ingin menyimpan 100 nama siswa. Tanpa array, kita harus membuat 100 variabel berbeda. Dengan array, cukup satu.

let siswa = ["Andi", "Budi", "Cici", "Dedi"];
console.log(siswa[0]); // Andi
console.log(siswa[1]); // Budi
console.log("Jumlah siswa: " + siswa.length); // 4
Latihan 4.1

Buat array berisi 5 nama makanan favorit. Tampilkan makanan pertama, makanan ketiga, dan jumlah elemen array.

let makanan = ["Nasi Goreng", "Mie Ayam", "Sate", "Bakso", "Soto"];
console.log("Makanan pertama: " + makanan[0]);
console.log("Makanan ketiga: " + makanan[2]);
console.log("Jumlah makanan: " + makanan.length);

4.2 Menambah Data ke Array

Ada dua cara menambah data: via indeks langsung atau menggunakan method push().

Mengisi via indeks

let buah = ["Apel", "Mangga"];
buah[2] = "Jeruk"; // menambah di indeks 2
console.log(buah); // ["Apel", "Mangga", "Jeruk"]

Menggunakan push()

let sayur = ["Wortel", "Bayam"];
sayur.push("Tomat"); // tambah satu
sayur.push("Terong", "Timun"); // tambah banyak
console.log(sayur); // ["Wortel", "Bayam", "Tomat", "Terong", "Timun"]
Latihan 4.2

Buat array kosong let hewan = []. Tambahkan "Kucing", "Anjing", "Ayam" menggunakan push(). Lalu tambahkan "Sapi" di indeks 3. Tampilkan semua isi array.

let hewan = [];
hewan.push("Kucing");
hewan.push("Anjing");
hewan.push("Ayam");
hewan[3] = "Sapi";
console.log(hewan); // ["Kucing", "Anjing", "Ayam", "Sapi"]

4.3 Menghapus Data dari Array

Dua cara umum: delete (tidak disarankan) dan pop() (disarankan).

Menggunakan delete

let buah = ["Apel", "Mangga", "Jeruk"];
delete buah[1];
console.log(buah); // ["Apel", empty, "Jeruk"] (panjang tetap 3)

Menggunakan pop()

let buah = ["Apel", "Mangga", "Jeruk"];
let terhapus = buah.pop(); // menghapus "Jeruk"
console.log(buah); // ["Apel", "Mangga"]
console.log(terhapus); // "Jeruk"
Latihan 4.3

Buat array berisi [10, 20, 30, 40]. Hapus elemen terakhir dengan pop(), lalu tampilkan elemen yang dihapus dan array sekarang.

let angka = [10, 20, 30, 40];
let dihapus = angka.pop();
console.log("Elemen dihapus: " + dihapus); // 40
console.log("Array sekarang: " + angka); // [10, 20, 30]

4.4 Array dan Perulangan

Kita bisa menggabungkan array dengan perulangan untuk memproses semua elemen.

<ul id="daftar"></ul>
<script>
    let produk = ["Senter", "Radio", "Antena", "Obeng"];
    let teks = "";
    for (let i = 0; i < produk.length; i++) {
        teks += "<li>" + produk[i] + "</li>";
    }
    document.getElementById("daftar").innerHTML = teks;
</script>

Hasilnya akan menampilkan daftar produk dalam bullet list.

Latihan 4.4

Buat array berisi 5 mata pelajaran. Tampilkan dalam bentuk daftar bernomor (<ol>) menggunakan perulangan.

<ol id="daftar"></ol>
<script>
    let mapel = ["Matematika", "Fisika", "Kimia", "Biologi", "Bahasa Inggris"];
    let teks = "";
    for (let i = 0; i < mapel.length; i++) {
        teks += "<li>" + mapel[i] + "</li>";
    }
    document.getElementById("daftar").innerHTML = teks;
</script>

MARKICOB – Eksperimen Array

Coba kode berikut untuk melihat bagaimana array dapat dimodifikasi dengan berbagai cara.

// Program sederhana: daftar belanja
let belanja = ["Susu", "Roti", "Telur"];
console.log("Daftar belanja awal:", belanja);

// Tambah item
belanja.push("Mentega");
console.log("Setelah push:", belanja);

// Hapus item terakhir
belanja.pop();
console.log("Setelah pop:", belanja);

// Ubah item di indeks tertentu
belanja[1] = "Roti Gandum";
console.log("Setelah ubah:", belanja);
