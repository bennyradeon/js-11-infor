Bab 3: Struktur Kontrol

Percabangan (if/else, switch) dan perulangan (for, while, do while).

MARKICOB – Coba Sendiri!

Sebelum memulai, coba jalankan kode berikut di console browser (F12) atau buat file HTML baru. Amati apa yang terjadi!

// Program sederhana untuk menentukan ganjil/genap
let angka = prompt("Masukkan sebuah angka:");
angka = Number(angka);
if (angka % 2 === 0) {
    console.log(angka + " adalah bilangan genap.");
} else {
    console.log(angka + " adalah bilangan ganjil.");
}
// Coba jalankan beberapa kali dengan angka berbeda.
// Modifikasi program untuk menampilkan pesan dengan alert.

Petunjuk: Program ini menggunakan if-else dan operator modulus. Ini akan membantu kalian memahami percabangan.

3.1 Percabangan (Conditional)

Dalam hidup, kita sering mengambil keputusan: jika hujan, bawa payung; jika tidak, tidak usah. Di pemrograman, kita menggunakan percabangan.

3.1.1 if, else if, else

if digunakan untuk menjalankan kode jika kondisi benar. else untuk alternatif. else if untuk kondisi tambahan.

let nilai = 85;
if (nilai >= 90) {
    console.log("Nilai A");
} else if (nilai >= 75) {
    console.log("Nilai B");
} else {
    console.log("Nilai C");
}

Hasilnya akan "Nilai B" karena 85 >= 75.

3.1.2 switch

switch digunakan ketika banyak pilihan berdasarkan satu nilai. Cocok untuk menu atau hari.

let hari = "Senin";
switch (hari) {
    case "Senin":
        console.log("Hari belajar JavaScript");
        break;
    case "Selasa":
        console.log("Hari belajar CSS");
        break;
    default:
        console.log("Hari libur");
}

Jangan lupa break agar tidak menjalankan case berikutnya.

Latihan 3.1
  1. Buat program yang meminta pengguna memasukkan angka (gunakan prompt), lalu menentukan apakah angka tersebut positif, negatif, atau nol. Tampilkan hasilnya dengan alert.
  2. Buat program dengan switch untuk menampilkan nama bulan berdasarkan nomor bulan (1 = Januari, 2 = Februari, ..., 12 = Desember). Tampilkan hasilnya dengan alert.
// Jawaban Latihan 3.1
// 1. Positif/Negatif/Nol
let angka = prompt("Masukkan angka:");
angka = Number(angka);
if (angka > 0) {
    alert("Angka positif");
} else if (angka < 0) {
    alert("Angka negatif");
} else {
    alert("Angka nol");
}

// 2. Nama bulan dengan switch
let bulan = prompt("Masukkan nomor bulan (1-12):");
bulan = Number(bulan);
let namaBulan;
switch (bulan) {
    case 1: namaBulan = "Januari"; break;
    case 2: namaBulan = "Februari"; break;
    case 3: namaBulan = "Maret"; break;
    case 4: namaBulan = "April"; break;
    case 5: namaBulan = "Mei"; break;
    case 6: namaBulan = "Juni"; break;
    case 7: namaBulan = "Juli"; break;
    case 8: namaBulan = "Agustus"; break;
    case 9: namaBulan = "September"; break;
    case 10: namaBulan = "Oktober"; break;
    case 11: namaBulan = "November"; break;
    case 12: namaBulan = "Desember"; break;
    default: namaBulan = "Bulan tidak valid";
}
alert("Bulan: " + namaBulan);

3.2 Perulangan (Looping)

Perulangan digunakan untuk menjalankan kode berulang kali. Ada tiga jenis: for, while, do...while.

3.2.1 Perulangan for

for digunakan jika jumlah perulangan sudah diketahui. Terdiri dari inisialisasi, kondisi, dan increment/decrement.

for (let i = 1; i <= 5; i++) {
    console.log("Iterasi ke-" + i);
}

Hasilnya akan mencetak angka 1 sampai 5.

3.2.2 Perulangan while

while digunakan jika jumlah perulangan tidak pasti. Selama kondisi true, perulangan terus berjalan.

let i = 1;
while (i <= 5) {
    console.log("Angka " + i);
    i++;
}

3.2.3 Perulangan do...while

do...while mirip while, tapi kode dijalankan minimal satu kali karena pengecekan di akhir.

let i = 1;
do {
    console.log("Angka " + i);
    i++;
} while (i <= 5);
Latihan 3.2
  1. Tampilkan bilangan genap dari 2 sampai 20 menggunakan for.
  2. Gunakan while untuk menjumlah angka 1 sampai 10. Tampilkan hasil jumlahnya.
  3. Buat program yang terus meminta input password sampai benar (misal password = "123"). Gunakan do...while.
// Jawaban Latihan 3.2
// 1. Bilangan genap 2-20
for (let i = 2; i <= 20; i += 2) {
    console.log(i);
}

// 2. Jumlah 1-10 dengan while
let i = 1;
let jumlah = 0;
while (i <= 10) {
    jumlah += i;
    i++;
}
console.log("Jumlah = " + jumlah);

// 3. Password dengan do...while
let password;
do {
    password = prompt("Masukkan password:");
} while (password !== "123");
alert("Password benar!");

MARKICOB – Eksperimen Perulangan

Coba jalankan kode berikut di console atau file HTML. Modifikasi nilai dan lihat perubahannya.

// Menampilkan tabel perkalian 1-5
for (let i = 1; i <= 5; i++) {
    let baris = "";
    for (let j = 1; j <= 5; j++) {
        baris += i + "x" + j + "=" + (i*j) + "\t";
    }
    console.log(baris);
}
// Perhatikan bagaimana perulangan bersarang bekerja.
