Bab 5: Fungsi

Membuat kode yang dapat dipanggil berulang kali, dengan atau tanpa event.

MARKICOB – Coba Sendiri!

Jalankan kode berikut di console browser atau buat file HTML. Fungsi adalah resep yang bisa dipakai berulang.

// Membuat fungsi sederhana
function sapa(nama) {
    return "Halo, " + nama + "!";
}

// Memanggil fungsi
console.log(sapa("Budi"));
console.log(sapa("Siti"));

// Coba buat fungsi sendiri untuk menghitung luas persegi

Petunjuk: Fungsi membantu kita tidak menulis kode yang sama berulang kali.

5.1 Apa itu Fungsi?

Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu dan bisa dipanggil kapan saja. Ibarat resep masakan: kita tulis sekali, bisa dipakai berkali-kali.

Mengapa perlu fungsi?

  • Menghindari penulisan kode berulang.
  • Program lebih terstruktur.
  • Memudahkan perbaikan (cukup ubah di satu tempat).

Cara Mendeklarasikan Fungsi

Function Declaration:

function tambah(a, b) {
    return a + b;
}

Function Expression:

const kali = function(a, b) {
    return a * b;
};

Arrow Function (ES6):

const bagi = (a, b) => a / b;

Parameter dan Return

Fungsi bisa menerima input (parameter) dan mengembalikan output (return). Jika tidak ada return, fungsi mengembalikan undefined.

function sapa(nama) {
    console.log("Halo " + nama);
    // tidak ada return, jadi undefined
}
let hasil = sapa("Budi"); // console log, hasil = undefined
Latihan 5.1
  1. Buat fungsi luasSegitiga(alas, tinggi) yang mengembalikan luas segitiga (alas * tinggi / 2).
  2. Buat fungsi sapa(nama) yang mengembalikan "Halo, [nama]!".
  3. Panggil kedua fungsi tersebut dan tampilkan hasilnya di halaman HTML menggunakan innerHTML.
<!DOCTYPE html>
<html>
<head><title>Latihan Fungsi</title></head>
<body>
    <div id="hasil"></div>
    <script>
        function luasSegitiga(alas, tinggi) {
            return (alas * tinggi) / 2;
        }
        function sapa(nama) {
            return "Halo, " + nama + "!";
        }
        let hasil1 = luasSegitiga(10, 5);
        let hasil2 = sapa("Budi");
        document.getElementById("hasil").innerHTML = "Luas: " + hasil1 + "<br>" + hasil2;
    </script>
</body>
</html>

5.2 Fungsi dengan Event

Event adalah kejadian seperti klik tombol, mouse bergerak, atau keyboard ditekan. Kita bisa memanggil fungsi saat event terjadi.

Menggunakan atribut onclick (Inline)

<button onclick="tampilkanPesan()">Klik Saya</button>
<p id="pesan"></p>
<script>
    function tampilkanPesan() {
        document.getElementById("pesan").innerHTML = "Tombol diklik!";
    }
</script>

Menggunakan addEventListener (lebih disarankan)

<button id="tombol">Klik Saya</button>
<p id="pesan"></p>
<script>
    let tombol = document.getElementById("tombol");
    function tampilkanPesan() {
        document.getElementById("pesan").innerHTML = "Tombol diklik!";
    }
    tombol.addEventListener("click", tampilkanPesan);
</script>

Keuntungan addEventListener: bisa menambahkan banyak listener, mudah dihapus, dan memisahkan HTML dari JavaScript.

Latihan 5.2

Buat halaman dengan dua input angka dan tombol "Jumlah". Saat tombol diklik, panggil fungsi yang menjumlahkan kedua angka dan tampilkan hasilnya di halaman. Gunakan addEventListener.

<!DOCTYPE html>
<html>
<head><title>Kalkulator</title></head>
<body>
    <input type="number" id="a" placeholder="Angka 1">
    <input type="number" id="b" placeholder="Angka 2">
    <button id="hitung">Jumlah</button>
    <p id="hasil"></p>
    <script>
        let tombol = document.getElementById("hitung");
        function jumlah() {
            let angka1 = Number(document.getElementById("a").value);
            let angka2 = Number(document.getElementById("b").value);
            let hasil = angka1 + angka2;
            document.getElementById("hasil").innerHTML = "Hasil: " + hasil;
        }
        tombol.addEventListener("click", jumlah);
    </script>
</body>
</html>

MARKICOB – Eksperimen Fungsi dan Event

Coba buat halaman sederhana dengan tombol yang mengubah warna latar belakang. Gunakan fungsi dan event.

<button id="btnWarna">Ubah Warna</button>
<div id="kotak" style="width:100px;height:100px;background:red;"></div>
<script>
    let btn = document.getElementById("btnWarna");
    let kotak = document.getElementById("kotak");
    btn.addEventListener("click", function() {
        kotak.style.backgroundColor = "blue";
    });
</script>

Modifikasi agar warnanya berganti acak setiap kali diklik.
